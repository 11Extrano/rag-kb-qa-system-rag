## ADDED Requirements

### Requirement: 系统根据检索片段与用户问题生成答案

系统必须根据用户问题与检索到的片段列表（含可选相似度分数），使用配置的 LLM 或等价能力生成自然语言答案，记录或发出「答案生成完成」，并将答案返回给用户。

#### Scenario: 返回带内容的答案

- **WHEN** 用户已提交问题且检索上下文返回了至少一个匹配片段
- **THEN** 系统必须基于问题与片段内容生成答案并返回给用户，并标记「答案生成完成」

#### Scenario: 无匹配片段时的应答

- **WHEN** 检索上下文返回空片段列表
- **THEN** 系统仍须响应用户（如提示未找到相关内容或知识库为空），且不得在无依据的情况下由 LLM 编造内容

---

### Requirement: 系统支持引用来源

系统必须支持在响应中返回对来源片段的引用（如片段 id、文档 id 或摘要），以便用户知晓答案依据了知识库的哪些部分。

#### Scenario: 答案包含引用

- **WHEN** 系统返回基于一个或多个检索片段生成的答案
- **THEN** 响应必须包含对这些片段的引用（如文档名、片段 id 或短摘要），供用户核实来源

#### Scenario: 引用在响应格式中可选

- **WHEN** 客户端请求带引用的答案
- **THEN** 系统必须在响应中包含引用列表；若客户端不要求引用，系统可省略或简化
