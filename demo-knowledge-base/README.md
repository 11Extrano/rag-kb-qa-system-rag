# Demo 知识库

本目录为 RAG 知识库问答系统的示例文档，可直接上传到系统用于体验检索与问答。

## 文件说明

| 文件 | 内容概要 |
|------|----------|
| 人事制度.md | 年假、考勤、报销、人事联系方式 |
| 产品说明.md | 产品定位、支持平台、文档格式、核心功能与版本 |
| 开发规范.txt | 分支策略、提交规范、接口安全、测试与上线 |
| 常见问题.md | 账号登录、文档上传、检索与问答、技术支持 FAQ |

## 如何使用

1. **启动服务**：确保 MySQL、后端与（可选）前端已按项目 README 启动。
2. **上传文档**：通过管理端接口上传本目录下的文件。
   - 接口：`POST /api/admin/documents`
   - Content-Type: `multipart/form-data`，字段名：`file`（具体以服务端约定为准）
   - 可用 curl 示例（在项目根目录执行）：
     ```bash
     curl -X POST http://127.0.0.1:7001/api/admin/documents \
       -F "file=@demo-knowledge-base/人事制度.md"
     ```
3. **等待处理**：上传后系统会异步完成清洗、分块与向量化，稍等再提问。
4. **体验问答**：调用 `POST /api/qa`，body `{ "question": "公司年假几天？" }` 等，即可得到带引用的答案。

## 推荐试问示例

- 公司年假几天？年假上限多少？
- 产品支持哪些文档格式？单文件多大？
- 分支策略是什么？提交信息格式怎么写？
- 忘记密码怎么办？支持 SSO 吗？
