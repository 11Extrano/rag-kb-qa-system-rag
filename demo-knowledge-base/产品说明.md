# 产品说明

## 产品名称与定位

本产品名为「智能知识库问答系统」，面向企业内网文档，提供上传、检索与自然语言问答能力。

## 支持平台与部署

- **部署方式**：支持 Docker 部署，也可在 Linux / macOS 服务器上直接运行 Node 服务。
- **浏览器**：推荐 Chrome、Edge、Safari 最新版。
- **文档格式**：支持 .txt、.md、.pdf、.doc、.docx、.html 格式上传，单文件不超过 50MB。

## 核心功能

1. **文档管理**：管理员可上传、列表查看、删除文档。
2. **智能检索**：系统对文档做清洗、分块与向量化，支持语义检索。
3. **问答**：用户输入问题后，系统检索相关片段并生成带引用来源的答案。
