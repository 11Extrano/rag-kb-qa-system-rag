# 常见问题 FAQ

## 账号与登录

**Q：忘记密码怎么办？**  
A：在登录页点击「忘记密码」，通过注册邮箱收取重置链接。若邮箱不可用，请联系 IT 支持（it@company.com）。

**Q：支持 SSO 单点登录吗？**  
A：支持。企业版已对接 LDAP/SSO，需管理员在后台配置认证源。

## 文档与知识库

**Q：上传的文档会做内容审核吗？**  
A：目前仅做格式与大小校验，不做自动内容审核。请勿上传涉密或违规内容。

**Q：单次最多能问多少字？**  
A：提问建议控制在 500 字以内，以保证检索与回答质量。答案长度由系统根据检索结果与模型配置决定。

**Q：检索不到相关内容时怎么办？**  
A：系统会尽量基于已有知识库回答；若库中无相关内容，会提示「未找到相关文档」。可尝试换一种问法或补充上传相关文档。

## 技术支持

- 文档与使用问题：support@company.com
- 故障与报错：先查看 run 目录下的日志，仍无法解决时提交工单并附上错误截图或日志片段。
